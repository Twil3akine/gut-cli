# gut

`gut` is a tiny joke CLI for people who type `gut` when they meant `git`.

It prints a small goose:

```text
 _
__(.)<
/___)
 " "
```

Then it makes fun of your typo in English.

## Install

### From source

```bash
cargo install --path .
```

### From crates.io

```bash
cargo install gut
```

## Run

```bash
gut
```

## Why this exists

Because mistyping `git` deserves immediate consequences.
